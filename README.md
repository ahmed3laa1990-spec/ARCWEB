## 📁 هيكل الملفات

```
new-site/
├── index.html         ← الموقع كاملاً (HTML + CSS + JS في ملف واحد)
├── 404.html           ← صفحة الخطأ المخصصة
├── robots.txt         ← إرشادات محركات البحث
├── sitemap.xml        ← خريطة الموقع للـ SEO
├── _headers           ← أمان + كاش (Cloudflare Pages)
├── _redirects         ← تحويلات URL (Cloudflare Pages)
├── .gitignore         ← ملفات يتجاهلها Git
├── assets/            ← الصور واللوجو
├── DEPLOY-NOW.md      ← دليل النشر الكامل
├── EDITING-GUIDE.md   ← دليل التعديل اليومي
└── README.md          ← هذا الملف
```
